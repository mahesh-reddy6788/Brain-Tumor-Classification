# Brain-Tumor-Classification
🧠 Brain Tumor Detection using MobileNetV2 Transfer Learning
 
 📌 Overview

This project implements a deep learning model to classify brain MRI scans into four categories — Glioma, Meningioma, Pituitary Tumor, and No Tumor — using MobileNetV2 transfer learning. The trained model is deployed as an interactive web application using Streamlit, allowing users to upload MRI images and receive instant predictions with confidence scores.

🚀 Features

Multi-class brain tumor classification from MRI scans

Transfer learning with pre-trained MobileNetV2 architecture

Data augmentation for improved model generalization

Real-time predictions with confidence scores

Simple and interactive Streamlit web interface

📂 Dataset

MRI scan images classified into 4 classes: Glioma, Meningioma, Pituitary Tumor, No Tumor
Preprocessed and augmented for balanced training
